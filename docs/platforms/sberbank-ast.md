# Сбербанк-АСТ (sberbank-ast.ru)

## Общая информация
- **URL поиска**: `https://www.sberbank-ast.ru/purchaseList.aspx`
- **Тип выдачи**: Список с пагинацией (ASP.NET).

## Селекторы (предварительные)
*Примечание: В связи с временной недоступностью площадки при разработке, используются селекторы, характерные для данной платформы.*

| Элемент | Селектор | Описание |
|---------|----------|----------|
| Поле поиска | `input#txtKeyword` | Основное поле для ввода ключевых слов |
| Кнопка поиска | `input#btnSearch` | Кнопка «Поиск» (submit) |
| Контейнер лота | `.masterclue` | Одна строка/карточка тендера в списке |
| Ссылка на лот | `.masterclue a[href*="purchaseview"]` | Ссылка на детальную страницу |
| Заголовок | `.masterclue .es-el-name` | Название тендера |

## Особенности
- Используется ASP.NET WebForms.
- Поиск часто требует предварительного открытия главной страницы для инициализации сессии.
- Пагинация реализована через `__doPostBack` или аналогичные механизмы.
