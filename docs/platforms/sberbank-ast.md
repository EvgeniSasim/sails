# Сбербанк-АСТ (sberbank-ast.ru)

См. общую стратегию: [extraction-strategy.md](./extraction-strategy.md).

## Поиск и навигация (семантика, не CSS)

| Действие | Как находим |
|----------|-------------|
| Поле ключевых слов | `get_by_placeholder(/Введите запрос/i)` |
| Кнопка поиска | `get_by_role("button", name=/Поиск/i)` |
| Результаты загружены | текст «Найдено процедур» или наличие `leaf:objectHrefTerm` |
| Cookie | `get_by_role("button", name="Принять")` |
| Следующая страница | кнопка/элемент с текстом `>` |
| Готовность страницы | ожидания `wait_page_idle` (скрытие overlay) |

## Данные выдачи (leaf-поля)

В каждом блоке процедуры читаем атрибуты `content`:

| Leaf | Поле |
|------|------|
| `leaf:objectHrefTerm` | URL карточки |
| `leaf:purchName` | Название |
| `leaf:purchAmount` | Сумма |
| `leaf:purchCurrency` | Валюта |

Номер процедуры — из текста блока по шаблону `№ <цифры>`.

## Карточка тендера (текст по меткам)

Из `main.innerText`:

- `Сведения о закупке № …`
- `Наименование объекта закупки`
- `Сведения об организаторе торгов`
- `Начальная (максимальная) цена контракта`
- `Дата и время окончания срока подачи заявок`
- `Контактная информация`
- `… Публикация извещения` (дата публикации)

## Особенности

- **Никаких CSS-селекторов** для данных в коде адаптера.
- URL карточки может быть с префиксом пути: `/ZK/purchaseview.aspx?id=…`.
- Фильтр периода — на шаге detail, если дата публикации распознана в тексте.
